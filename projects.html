Code Vigilant

Code Vigilant project is created out of the need to have a more secure open source software. It is a known fact that a large number of users use opensource software but a very few of them contribute back in terms of identifying and making these opensource software a more secure piece of software. As part of the initial experiment found multiple vulnerabilities in wordpress plugins. The next phase we are focusing on multiple softwares.

PAC

PAC - Palo Alto Command line utility. This project is created to make Palo Alto firewall adminsitrators life easier by automating things like URL Filtering, PAN-DB, Security Policy, etc. This framework is based on PAN-CLI features and using paramiko to make connections & execute commands. It's written in Python.

Security-Headers

This is wrapper around Security Headers. Basic shell script for automating using API call with JSON parsing. It gives simple command line interface for Security Headers for any website.
